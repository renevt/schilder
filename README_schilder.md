## 🚀 Hoe deployen

### Via GitHub Pages
1. Push HTML naar `main` branch
2. Zet `index.html` als de hoofdpagina
3. GitHub Pages gaat live op: `https://renevt.github.io/schilder`

1. Voeg domain in bij GitHub Pages settings
2. Zet DNS CNAME record naar GitHub
3. GitHub genereert automatisch HTTPS certificaat
